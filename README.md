📱 Termux Installation Commands

📲 Step by Step

```bash
pkg update && pkg upgrade -y
pkg install -y python python-pip git cloudflared openssl-tool
git clone https://github.com/shahid2005a/DGTL-CAM.git
pip install flask colorama
cd DGTL-CAM
python Main.py
```


🛠️ Troubleshooting

Issue Solution
❌ python: command not found Reinstall Python: pkg install python
❌ pip: command not found Install pip: pkg install python-pip
❌ Permission denied Check storage permissions
❌ No module 'flask' Run pip install flask colorama
❌ Tunnel not working Check internet, wait 30s
❌ Port in use Change port in script
❌ git: command not found Run pkg install git
❌ cloudflared: not found Run pkg install cloudflared
❌ openssl: not found Run pkg install openssl-tool
